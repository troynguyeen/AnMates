## 2. Typography Audit

### 2.1 Font Families

| Role | Design spec | Code | Status |
|---|---|---|---|
| Display/Heading | Plus Jakarta Sans | `GoogleFonts.plusJakartaSans` | ✅ Match |
| Body/Chat | Be Vietnam Pro | `GoogleFonts.beVietnamPro` | ✅ Match |
| Mono/Caption | JetBrains Mono | `GoogleFonts.jetBrainsMono` | ✅ Match |

### 2.2 Type Scale Deviations

| Level | Design (size/weight/lh) | Code (`AppTextStyles`) | Deviation |
|---|---|---|---|
| H1 | 32px / w700 / 1.2 | `heading1()` = 30px / w800 / 1.1 | ⚠️ -2px size, +1 weight level, lh off |
| H2 | 24px / w600 / 1.3 | `heading2()` = 24px / w800 / 1.15 | ⚠️ +2 weight levels, lh off |
| H3 | 18px / w600 / 1.4 | `display(size:18, weight:w700)` | ⚠️ +1 weight level |
| Body | 16px / w400 / 1.5 | `body(size:15)` | ⚠️ -1px size |
| Body small | 14px / w400 / 1.5 | `body(size:14)` | ✅ Match |
| Caption | 12px / w400 / 1.4 | `mono(size:11)` | ⚠️ -1px size, font switch |
| Button | 14px / w600 | `cta()` = 17px / w700 | ⚠️ **Major** — +3px, +1 weight level |

> **H1 fix:** Change `heading1()` → `size: 32, weight: FontWeight.w700, height: 1.2, letterSpacing: -1.0`
> **CTA fix:** Change `cta()` → `size: 14, weight: FontWeight.w600`

## 3. Deprecated API Usage

`withOpacity()` is deprecated in Flutter 3.x — must replace with `.withValues(alpha:)`.

### Occurrences:

| File | Count | Example |
|---|---|---|
| `anm_widgets.dart` | 9 | `AppColors.berry.withOpacity(0.35)` → `.withValues(alpha: 0.35)` |
| `splash_screen.dart` | 2 | `Colors.white.withOpacity(0.2)` |
